What is LLAMA First Aid?

Live Assistant for Medical Assistance

  • AI-Powered Assistant

    Provides real-time guidance during medical emergencies with step-by-step instructions

  • Built with Meta's LLaMA

    Leverages advanced language model for responsive, intelligent assistance

  • Dual Input Methods

    Text and audio options for ease of use in high-stress situations

  • Location-Based Recommendations

    Suggests nearest hospitals and medical facilities based on user location

  • Designed for Bangladesh

    Specifically tailored to address healthcare gaps in rural and urban areas

Technical Architecture

How LLAMA First Aid Works

Multi-Agent Approach

  • Triage Agent: Assesses emergency severity
  • Specialized Agents: Provide domain-specific assistance
  • LangGraph Orchestration: Coordinates the agent workflow
  • Data Storage: GCP Bucket for secure information management
  • User Interface: Streamlit for accessible interaction
